Focused Image Comparison: Rendered vs Reference

This page shows detailed comparisons focusing on key differences between rendered and reference images.

Summary of Findings

The main differences between rendered and reference images are:

  1. Image Dimensions: Rendered images are 99x16 pixels while reference images are 100x16 pixels.
  2. Trailing spaces: The text representations show some lines with slightly different trailing spaces.
  3. Content Differences: For special cases like album display and alerts, the content is completely different between rendered and reference.

These differences explain why exact matching is failing, but they don't indicate actual bugs in the rendering - just differences in how the content is displayed or trimmed.
